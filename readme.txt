== Description ==

Image scroll is flexible Wordpress plugin that lets you create vertical sliders for your images. You can choose every image in your media or upload new one. You can choose size of displayed images, enable lightbox for this images. Adding shortode to page content is very simple, you just click on button in editor.

== Installation ==

1. Upload "Image Scroll" folder to the "/wp-content/plugins/" directory
2. Activate the plugin through the "Plugins" menu in WordPress
3. Create new or go to exiting posts/page and use tinymce editor's button for create image scroll shortcode.
